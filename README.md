# Advent of Code 2025

My solutions to Advent of Code 2025 puzzles, implemented in C#/.NET.

---

# How to Use

Follow these steps to build and run the solutions for a specific day:

## 1. Download repository

- Obtain the code using Git: `git clone https://github.com/OpaliteArchitect/Advent_Of_Code_2025.git`

## 2. File Input: 

- Download your personal puzzle input from Advent of Code as raw text file (no .txt extension).
- Locate the appropriate folder for the puzzle day you wish to run. 
- Delete the existing raw text file with the format "DayXInput".
- Add your personal puzzle input into this folder.
- The file must be named "DayXInput", with X representing the day of the puzzle (e.g: Day1Input, Day2Input).

## 3. Code Activation: 

- Open Program.cs and locate the comment block corresponding to the puzzle day you are testing.
- Uncomment the entire block.

## 4. Build and Run:

- Build the project using the .NET CLI or your IDE (e.g., dotnet build).
- Run the project (e.g., dotnet run).

The calculated answers will be displayed in the console output.
